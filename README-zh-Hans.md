# FreeTSDB 

## An Open-Source Time Series Database, and the BEST Open-Source replacement for InfluxDB Enterprise.

FreeTSDB是InfluxDB企业版的开源替代，增强了代码健壮性、补齐集群等高可用能力、100%兼容InfluxDB生态和接口。
